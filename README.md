# unmult.anm
[【AVIUtl】黒背景の素材を半透明にするスクリプト【unmult】](https://www.nicovideo.jp/watch/sm39935396) のDLL版。

## インストール

右のReleasesからdllとanmをダウンロードし、 `/Plugins/script` に置いてください。

## ライセンス

MITライセンスで公開しています。
AviUtlで使用する場合もクレジット表記は不要です。
